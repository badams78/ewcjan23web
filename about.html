Overview

This taxonomy was derived using embedding-based clustering, with no predefined schema imposed. The structure emerges from the content patterns in the article corpus.

Clustering Method

Primary Clustering: Agglomerative clustering with Ward linkage on article embeddings. The optimal number of clusters (10-15) was determined by silhouette score optimization.

Subclustering: Each primary cluster is further divided into subclusters using the same method. The number of subclusters varies per primary based on internal diversity (2-20 subclusters, minimum 15 articles each).

Visualizations (v2)

This enhanced version includes four interactive visualizations on each subcluster page:

  • Similarity Heatmap: 70-cell row showing similarity to all subclusters
  • Spider Chart: Radar chart showing relationship to 15 primary clusters
  • 2D Embedding Map: t-SNE projection of all subclusters
  • Network Graph: Force-directed graph of connections above threshold

LLM Enrichment

Cluster names and descriptions were generated by Claude (claude-sonnet-4-20250514) based on sample articles from each cluster. No predefined categories were imposed.
